Introduction

Mela Fast Builder is the fron-end framework which can be used to speed up the development process. In nutshell mela is predefined CSS and JAVASCRIPT which can be edited so as to fit the developers needs. Developer can deploy mela he/she just need to include two files which are mela.css and mela.js and that is all what is required.

index.html

<!DOCTYPE html> <html> <head> <title></title> <link rel="stylesheet" type="text/css" href="mela/mela.css"> <script src="mela/mela.js"></script> </head> <body> <nav></nav> </body> </html>
