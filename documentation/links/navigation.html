Navigation

Mela Fast Builder offers ways to make navigation you can either use navigation tag or use div with id or class name of "nav" it will bring the same desired ouput. Mela will bring the navigation with no background color but with a box shadow.
Also the first achor tag of unordered list can have a class with name of "websitename" this will make it unique by styling it with big font size

index.html

<!DOCTYPE html> <html> <head> <title></title> <link rel="stylesheet" type="text/css" href="mela/mela.css"> <script src="mela/mela.js"></script> </head> <body> <nav></nav> <!-- or you can use--> <nav id="nav"> <ul> <li><a href="#" class="websitename">WEBSITENAME</a></li> <li><a href="#">HOME</a></li> <li><a href="#">SERVICES</a></li> <li><a href="#">CONTACT</a></li> <li><a href="#">ABOUT</a></li> </ul> </nav> </body> </html>

Navigation contents

Inside navigation mela offers predefined styles for unordered <\li> with their respectively achor tags you can create responsive navigation menu which it's menu items will disappear when the width of a device will go below 800px

Menu button
